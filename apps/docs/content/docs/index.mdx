## What is Swashbuckler?

Swashbuckler is a flexible knowledge management tool that lets you create, organize, and connect your notes, documents, and ideas.

## Quick Start

1. Visit [swashbuckler.quest](https://swashbuckler.quest) to get started
2. Create a space to organize your content
3. Add objects using custom types and templates
4. Link related objects together with mentions and relations
