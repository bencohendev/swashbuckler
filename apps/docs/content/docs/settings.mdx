## Overview

The **Settings** page is the central hub for configuring Swashbuckler. It's split into two sections — **Account** settings that apply everywhere, and **Space** settings scoped to whichever space you're currently viewing.

Open Settings from the sidebar or by navigating to `/settings`.

{/* screenshot: Settings hub showing Account and Space sections */}

## Account Settings

These settings apply to your entire account, regardless of which space is active.

| Page | What It Does |
|------|--------------|
| **[Account](/docs/export)** | Profile info, password, and data export |
| **[Spaces](/docs/spaces)** | Create, rename, archive, and delete spaces |
| **[Custom Themes](/docs/themes)** | Create and manage custom color themes |

## Space Settings

These settings apply to the **current space** — the section heading shows the space name and icon so you always know which space you're configuring.

| Page | What It Does |
|------|--------------|
| **[Appearance](/docs/themes)** | Choose a theme for this space |
| **[Templates](/docs/templates)** | Create and manage entry templates |
| **[Types](/docs/entries-and-types)** | Add, edit, and reorder types and their properties |
| **[Sharing](/docs/sharing)** | Invite collaborators and manage permissions |

## Navigation

Each card in the hub links to a dedicated settings page. Use the **back arrow** or breadcrumb at the top of any settings page to return to the hub.
