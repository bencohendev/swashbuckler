# Sharing UI Redesign

**Status:** Done

## Summary

Redesign the sharing card/dialog for better usability, especially on desktop where the current layout feels cramped. Improve the overall look and feel of the share management interface.

## Motivation

The current sharing UI has layout issues — the email input, permission select, and invite button are squeezed together, and the shared user list doesn't scale well visually. Desktop has plenty of space that isn't being used effectively.

## Scope

- Wider dialog on desktop with better use of horizontal space
- Clearer visual hierarchy: invite form separate from shared user list
- Better spacing and alignment for the email input, role select, and invite button
- Shared user list with cleaner row layout (avatar, email, role, actions)
- Responsive — still works well on mobile (existing mobile fix preserved)
- No functional changes to sharing logic, permissions, or exclusions
